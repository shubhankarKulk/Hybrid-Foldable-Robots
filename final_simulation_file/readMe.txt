Make sure you have following files in your directory, in order to run the various examples:

1. vrep.py
2. vrepConst.py
3. the appropriate remote API library: "remoteApi.dll" (Windows), "remoteApi.dylib" (Mac) or "remoteApi.so" (Linux)
5. run the python script with the repositories and libraries previously installed.
6. after running the code in the background, start the simulation, otherwise the simulation will not function as desired
